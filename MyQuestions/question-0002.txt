Caso de uso: Eres analista en una inmobiliaria rural. El dataset contiene una variable ordinal calidad_construccion con categorías ["deficiente", "regular", "buena", "excelente"] que tiene un orden natural. También tienes variables numéricas como área y antigüedad. Quieres predecir el precio construyendo una nueva característica de interacción entre la calidad codificada y el área.
Tu Misión:
Escribe una función llamada regresion_con_interaccion(df, target_col, col_ordinal, categorias_orden) que realice lo siguiente:

Use OrdinalEncoder(categories=[categorias_orden]) de sklearn.preprocessing para transformar col_ordinal a valores numéricos enteros respetando el orden.
Reemplace la columna original col_ordinal en el DataFrame con el valor codificado.
Use numpy para crear una nueva columna interaccion multiplicando la columna codificada por la primera columna numérica restante (distinta a col_ordinal y target_col).
Separe X e y usando target_col.
Divida en train/test (75%/25%, random_state=7) y entrene un LinearRegression.
Devuelva un diccionario con:

"r2": coeficiente de determinación en test (redondeado a 4 decimales)
"coef_interaccion": coeficiente del modelo para la columna interaccion (float)



Librerías: pandas, numpy, sklearn
